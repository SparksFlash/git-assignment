# Git Assignment

## Commands Used

```bash
git init
git status
git add .
git commit -m "initial commit"
git branch
git checkout -b feature/login
git checkout -b bugfix/login-error
git merge feature/login
git rebase main
git log --oneline
git push origin main
```
